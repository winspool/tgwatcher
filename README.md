# tgwatcher 

The Game Watcher

Check remotely, what a system at home is doing.
Example usage is an Android emulator running a game.

This needs far less bandwidth compared to a full blown remote control software


# Build with autotools
* autoreconf
* configure
* make
* sudo make install
